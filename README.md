# Bilibili 增强助手 (Bilibili Enhancement Assistant)

一个基于 Chrome Extension (Manifest V3) 开发的 Bilibili 浏览体验增强工具。提供关键词屏蔽、快捷拉黑、用户数据透视等功能，帮助你净化 B 站首页和评论区。

## 核心功能

*   **🚫 关键词屏蔽**：
    *   支持设置多个关键词（如“营销号, 标题党”）。
    *   自动扫描首页、搜索页、热门页的视频标题。
    *   **智能屏蔽**：不仅匹配标题，还会异步检查视频的 Tags，如果 Tag 命中关键词也会屏蔽。
    *   屏蔽后卡片高亮并覆盖“拉黑UP”快捷按钮。
*   **⚡ 快捷拉黑/解除**：
    *   在评论区、动态列表、排行榜、热门视频等出现用户名的地方，自动插入“拉黑/解除”按钮。
    *   无需进入个人主页即可快速管理黑名单。
*   **🔍 悬停数据预览**：
    *   **用户详情**：鼠标悬停在用户名上，显示粉丝数、视频数、平均时长及**词云**（基于最近视频标题生成）。
    *   **视频详情**：鼠标悬停在视频封面上，显示视频 Tags 和 AI 总结（如果有）。
    *   **快捷复制**：悬停窗支持一键复制 UID。
*   **📋 黑名单管理**：
    *   点击插件图标，可查看当前已拉黑的用户列表。

## 安装说明

由于本项目尚未发布到 Chrome 应用商店，请通过“开发者模式”安装：

1.  下载本项目代码到本地（点击 Code -> Download ZIP 并解压）。
2.  打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/`。
3.  打开右上角的 **“开发者模式” (Developer mode)** 开关。
4.  点击左上角的 **“加载已解压的扩展程序” (Load unpacked)**。
5.  选择本项目所在的文件夹（即包含 `manifest.json` 的文件夹）。

## 使用指南

1.  **设置关键词**：点击浏览器右上角的插件图标，在输入框中输入关键词（用逗号分隔），点击保存。刷新 B 站页面即可生效。
2.  **拉黑用户**：在评论区或视频列表，点击用户名旁边的“拉黑”按钮。
3.  **查看详情**：鼠标悬停在用户名或视频封面上等待 0.5 秒。

## 开发与贡献

欢迎提交 Issue 或 Pull Request 来改进这个项目！

*   **核心逻辑**：
    *   `content.js`: 负责页面 DOM 扫描、插入按钮、悬浮窗逻辑。
    *   `background.js`: 处理跨域 API 请求（获取用户信息、操作黑名单）。
    *   `popup.js`: 插件弹窗交互。
